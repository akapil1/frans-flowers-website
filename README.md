# 🌸 Fran’s Flowers Website Redesign

A modern, responsive website redesign for a locally owned flower shop, focused on improving usability, visual appeal, and customer experience.

🔗 **Live Demo:** https://akapil1.github.io/frans-flowers-website/

---

## 📌 Project Overview
This project reimagines the online presence of a small business, transforming a basic layout into a clean, visually engaging, and user-friendly website.

The goal was to simulate a real-world client scenario and design a site that:
- highlights products clearly
- improves navigation
- enhances visual branding
- creates a welcoming customer experience

---

## ✨ Key Features
- Responsive layout for mobile and desktop
- Clean and modern UI design
- Improved navigation structure
- Dedicated About page for business storytelling
- Styled product/content sections for clarity

---

## 🛠 Tech Stack
- HTML5
- CSS3 (custom styling, layout, responsiveness)

---

## 🎯 What I Focused On
- Designing for a **real business use case**
- Improving **user experience and readability**
- Creating a **professional and approachable visual style**
- Structuring content to guide user flow
